âš™ï¸ How It Works

1

Data Ingestion

Product data is loaded from CSV files containing information about laptops, mobiles, and headphones with specifications, prices, and features.

2

Text Processing

Product descriptions are processed to create a weighted corpus where important fields (name, brand, specs) are given higher importance.

3

TF-IDF Indexing

A TF-IDF matrix is built using scikit-learn with optimized parameters for n-gram analysis and stop word removal.

4

Query Processing

User queries are analyzed for price constraints, category hints, and converted to TF-IDF vectors for similarity matching.

5

Result Ranking

Cosine similarity scores are calculated and results are filtered, ranked, and presented with relevance scores.
